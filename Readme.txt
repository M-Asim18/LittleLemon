API Endpoints to Test (Insomnia):
-----------------------------------
1. User Registration: POST /auth/users/
2. User Login (Get Token): POST /auth/token/login/
3. Get Menu Items: GET /restaurant/menu-items/
4. Add Menu Item (Auth Required): POST /restaurant/menu-items/
5. Get Single Item: GET /restaurant/menu-items/{id}/
6. Book a Table (Auth Required): POST /restaurant/booking/tables/
